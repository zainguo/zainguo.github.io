Android Studio 入门（一） 简介和安装

  • android
  • android studio

1.AndroidStudio 介绍

1.1.介绍 Android Studio 是一个Android开发环境，基于IntelliJ IDEA，Google在前年的I/O大会上推出后开始普及。
1.2.为什么要学习AS？
    1.1.1 Eclipse的ADT开发插件已经不再更新
    1.1.2 有些GitHub的开源库只提供AS的工程代码
1.3.与Eclipse的比较
    1.3.1AndroidStudio是Google官方推出的，专门为Android量身订做的一个开发工具，以后肯定会越来越强大;
    1.3.2速度更快(在大项目上体现明显，电脑配置越高越明显);
    1.3.3UI更漂亮(自带了黑色的炫酷主题)
    1.3.4更加智能的提示补全等功能
    1.3.5整合了Gradle构建工具，Gradle集合了Ant和Maven的优点，不管是配置、编译、打包都非常棒
    1.3.6强大的UI编辑器，附带多设备的实时预览
    1.3.7内置终端，即命令行输入
    1.3.8 目录结构和Eclipse不一样
    1.3.9 拥有强大、方便的插件

2.AndridStudio的安装

2.1 安装包版本介绍
    2.1.1 下载地址：
        官方的下载地址：http://developer.android.com/sdk/index.html
        国内的下载地址：http://www.androiddevtools.cn
    2.1.2 带SDK和不带SDK版
    2.1.3 安装版和解压版
2.2 展示安装过程
    2.2.1 AS的安装路径和工程路径都不要包含中文
2.3 第一次打开需要配置
    2.3.1 在Android Studio安装目录下的 bin 目录下，找到 idea.properties 文件，在文件最后追加 disable.android.first.run=true，避开连接谷歌服务器更新SDK
2.4 可能出现的问题
    没有配置JAVA_HOME 环境变量
